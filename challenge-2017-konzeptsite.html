Infos zur Challenge Konzeptsite

Die Konzeptsite soll zwei Funktionen erfüllen: eine anschauliche und interessante Vorstellung und Erklärung Ihres Features/Konzepts und eine kurze Dokumentation Ihrer Arbeit. Sie können diese beiden Teile auch auf zwei Unterseiten verteilen. Bei der Erstellung dieser Seite ist so gut wie alles erlaubt: Nutzung von Frameworks, Verwendung von Templates, Einbindung von Filmen, etc.

Vorstellung und Erklärung des Features

Erklären Sie Ihre Idee, bzw. Ihr Feature ausführlich und anschaulich. Erarbeiten Sie ein interessantes Layout. Nutzen Sie gute Bilder oder Illustrationen. Nutzen Sie ggf. animiertes Material. Machen Sie es dem User so einfach wie möglich Ihr Konzept zu verstehen.

Dokumentation

Erläutern Sie, wie Sie vorgegangen sind, welche Frameworks, Templates & größere Code Snippets Sie verwendet haben. Beschreiben Sie kurz, was die größten Schwierigkeiten waren. Weisen Sie uns ggf. auf halbfertige Code Snippets oder Branches hin. Kurz: machen Sie es uns so einfach wie möglich Ihre Arbeit fair zu bewerten.

Deployment

Bitte stellen Sie Ihre Konzeptsite via GitHub.io im Internet bereit. Denken Sie daran, dass Sie nur eigenes oder lizenzfreies Material verwenden. Fügen Sie einen Hinweis ein, dass es sich hier um ein studentisches Projekt handelt und nennen Sie sich als Autor. Wie das Deployment auf GitHub.io funktioniert, wird unten im Film erläutert.
